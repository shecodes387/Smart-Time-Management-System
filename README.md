# ⏰ Smart Time Management System

A C++ console-based Time Management System that provides multiple time-related utilities through a simple menu-driven interface.

## Features

- Digital Clock (12-Hour & 24-Hour)
- Date & Day Display
- Time Zone Support
- Stopwatch
- Countdown Timer
- Calendar
- Reminder System

## Technologies Used

- C++
- Windows Console API

## How to Run

1. Open `src/main.cpp` in your C++ IDE.
2. Compile the program.
3. Run the executable.
